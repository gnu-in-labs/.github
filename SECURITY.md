# Politique de Sécurité

## Versions Supportées

| Version | Supportée |
| ------- | ------------------ |
| v1.0.x  | ✅ Oui              |
| < v1.0  | ❌ Non              |

## Signaler une vulnérabilité

Merci de ne pas ouvrir d'issue publique pour une vulnérabilité.
Envoyez un e-mail à `security@gnu6.live` ou `admin@gnu6.live`.
Nous nous engageons à vous répondre sous 48 heures.
